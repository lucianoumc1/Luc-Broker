Ejemplo de Integración del Formulario de Cotización

Formulario Embebido

A continuación se muestra el formulario de cotización embebido en un iframe. Este formulario puede ser integrado en cualquier sitio web.

Código de Integración

Para integrar este formulario en tu sitio web, copia y pega el siguiente código HTML:

<!-- Formulario de Cotización de Seguros --> <div style="border: 1px solid #e5e7eb; border-radius: 8px; overflow: hidden; margin: 20px 0;"> <iframe src="https://tu-dominio.com/cotizacion-iframe.html" style="width: 100%; height: 100vh; min-height: 600px; border: none; display: block;" title="Formulario de Cotización de Seguros" allowfullscreen> </iframe> </div> <!-- Alternativa: Si necesitas un tamaño fijo --> <iframe src="https://tu-dominio.com/cotizacion-iframe.html" style="width: 100%; height: 700px; border: none; display: block;" title="Formulario de Cotización de Seguros"> </iframe>

Funcionamiento del JavaScript

✅ JavaScript Funciona Perfectamente

El archivo cotizacion.js se importa correctamente en el iframe:

  • El iframe carga su propio documento HTML completo
  • El script se ejecuta de forma independiente dentro del iframe
  • Todas las funcionalidades funcionan: selección de riesgo, validación, envío de datos
  • No hay conflictos con el JavaScript del sitio padre

Características del Formulario

  • ✅ Responsive: Se adapta automáticamente a diferentes tamaños de pantalla
  • ✅ Sin dependencias externas: Solo requiere Tailwind CSS (incluido via CDN)
  • ✅ Funcionalidad completa: Incluye todos los tipos de seguro disponibles
  • ✅ Validación: Campos obligatorios y validación de formulario
  • ✅ Envió de datos: Configurado para enviar datos a tu API
  • ✅ Diseño profesional: Interfaz moderna y atractiva

Configuración Requerida

⚠️ Importante

Antes de usar el formulario en producción, asegúrate de:

  • Actualizar la URL de la API en el archivo cotizacion.js
  • Configurar los valores correctos para typeOfOpportunity
  • Probar el envío de datos en un entorno de desarrollo
  • Verificar que el dominio donde se aloja el formulario esté en la lista blanca de CORS

Personalización

El formulario puede ser personalizado modificando:

  • Colores: Modifica las variables de color en la configuración de Tailwind
  • Tipos de seguro: Agrega o modifica los tipos de riesgo en riskFields
  • Campos: Personaliza los campos específicos de cada tipo de seguro
  • Mensajes: Cambia los textos y mensajes del formulario
